### Step 2: Install Required Plugins

Open Obsidian and install:

- Meta-bind
- Spaced Repetition
- TaskNotes
- Advanced Slides

---

### Step 3: Import Class

```bash
python import_class.py \
  --package TCM_101.zip \
  --student-id your_name
```

---

### Step 4: Start Learning

Open your Obsidian vault and navigate to:
`Classes/TCM_101/Dashboards/your_name_dashboard.md`
