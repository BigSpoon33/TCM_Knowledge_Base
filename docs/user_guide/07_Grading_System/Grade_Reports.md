## ğŸ¯ Report Generation

### Command Line

```bash
# Individual student report
python auto_grader.py \
  --class-id TCM_101 \
  --student-id john_doe \
  --report \
  --output john_doe_report.pdf

# Class-wide report
python auto_grader.py \
  --class-id TCM_101 \
  --class-report \
  --output TCM_101_class_report.pdf

# CSV export
python auto_grader.py \
  --class-id TCM_101 \
  --export-csv \
  --output TCM_101_grades.csv

# Analytics
python auto_grader.py \
  --class-id TCM_101 \
  --analytics \
  --output TCM_101_analytics.pdf
```
