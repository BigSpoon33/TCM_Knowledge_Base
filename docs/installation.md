# Installation

## Stable release

To install obsidian-capsule-cli, run this command in your terminal:

```sh
uv add python-boilerplate
```

Or if you prefer to use `pip`:

```sh
pip install python-boilerplate
```

## From source

The source files for obsidian-capsule-cli can be downloaded from the [Github repo](https://github.com/BigSpoon33/obsidian_capsule_cli).

You can either clone the public repository:

```sh
git clone git://github.com/BigSpoon33/obsidian_capsule_cli
```

Or download the [tarball](https://github.com/BigSpoon33/obsidian_capsule_cli/tarball/master):

```sh
curl -OJL https://github.com/BigSpoon33/obsidian_capsule_cli/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd obsidian_capsule_cli
uv pip install .
```
